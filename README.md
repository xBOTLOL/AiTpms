AiTpms is a Bluetooth-based Tire Pressure Monitoring System (TPMS) app for iOS.

The app scans nearby TPMS sensor broadcasts and displays real-time tire pressure, temperature, and sensor battery status.

## Features
- Real-time tire pressure monitoring
- Tire temperature display
- TPMS sensor battery status
- Background monitoring support
- Clear visual alerts

## Permissions
- **Bluetooth**: Used to scan and receive TPMS sensor broadcast data.
- **Location**: Required by iOS to enable Bluetooth scanning.  
  The app does not collect, store, or track user location data.
